About 關於

Because tuning shouldn't be the hard part 調音不應該是最難的環節

Every other tuner app is built for guitarists. Using one on a harp means scrolling through the wrong strings, reading the wrong notes, and second-guessing yourself. Harpie knows your harp — lever or pedal — so you can just tune and play. 其他所有調音 App 都是為吉他手設計的。Harpie 認識你的豎琴，無論是槓桿豎琴還是踏板豎琴，打開就能調，調完就能彈。

Features 功能

Auto Detection 自動偵測

Just pluck — it knows what to do 撥弦就好，其他交給它

No setup. No list to scroll through. Just pluck a string and the needle tells you exactly where you are. 不需要設定，不需要翻找清單。撥一根弦，指針告訴你現在在哪。

Reference Mode 參考模式

Tap a string — hear it, then tune to it 輕點一根弦，聽它的音，再調準它

Tap any string to hear what it should sound like, then tune to match. Ear and eye, at the same time. 輕點任意一根弦，聽它應該是什麼聲音，再調準。耳朵和眼睛同時並用。

Dark Mode 深色模式

Easy on the eyes in any light 任何光線下都舒適好看

Full dark theme for dim rehearsal rooms and pit orchestras. Switch anytime — your preference is saved. 完整深色主題，適合昏暗排練室與管弦樂坑。隨時切換，偏好自動保留。

FAQ 常見問題

What types of harps does Harpie support? Harpie 支援哪些類型的豎琴？

Harpie supports lever harp (19–40 strings, configurable to match your instrument) and pedal harp (full 47-string layout, C♭1–G♭7, all pedals set to flat). Each layout is built in — no manual setup needed. Harpie 支援槓桿豎琴（19–40 弦，可依樂器調整）與踏板豎琴（完整 47 弦，C♭1–G♭7，所有踏板從降音開始）。每種配置均已內建，無需手動設定。

How is Harpie different from a generic chromatic tuner app? Harpie 與一般半音調音 App 有什麼不同？

Generic tuner apps are built for guitars. Harpie has the full string layout for lever and pedal harps built in, a reference mode where you can tap any string to hear its correct pitch, an arc gauge that makes sharp/flat obvious at a glance, and A4 calibration from 430–450 Hz. Every detail is designed around how harpists actually tune. 一般調音 App 是為吉他設計的。Harpie 內建槓桿豎琴與踏板豎琴的完整弦列，參考模式可輕點任意弦播放正確音高，弧形音準儀讓偏高偏低一目了然，A4 校準範圍 430–450 Hz。每個細節都圍繞豎琴手的調音習慣而設計。
